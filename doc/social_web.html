API

These are the REST api documentation of the user discovery service. This is used to find the data service location for a user. The project source is hosted in https://github.com/ashrafuzzaman/social_web. And the service is hosted in http://social-web.heroku.com
Register a new user
Path /users.json
HTTP method POST
Params
email the email of the user
password password to authenticate
data_service_host the location url of the data service
Description Returns an auth token to use authenticated services
Signin for user
Path /api/sign_in.json
HTTP method POST
Params
email the email of the user
password password to authenticate
Description Returns an auth token to use authenticated services
Update data service host
Path /api/update_data_service_host.json
HTTP method POST
Params
email the email of the user
auth_token the auth token that is generated while login
data_service_host the new service location
Description Returns an auth token to use authenticated services
List of data service hosts for given emails
Path /api/data_service_hosts.json
HTTP method POST
Params
emails the list of email addresses of the people to search in the network
Description Returns list of data service hosts for given emails. This is used to find people in the network.
